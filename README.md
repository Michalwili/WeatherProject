<h1>WeatherApp</h1>

<p>A simple responsive weather app using api from Openweathermap.org, with background adjusting to search result. You can check the weather by typing the city name.</p>

<h4>Used technologies</h4>
<ul>
  <li>HTML5</li>
  <li>CSS3</li>
  <li>Bootstrap 4.0</li>
  <li>Javascript</li>
</ul>
